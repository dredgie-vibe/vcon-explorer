About this dashboard

This is the vCon Explorer — a static dashboard for the IETF vCon corpus stored alongside it in the same folder.

How it works

  • On load, the page fetches index.json and then each vcon-*.json file it lists.
  • Filter, search, and drill-down all run client-side. No server needed.
  • For audio (.wav) and video (.mp4), the player uses the url field on each dialog. If the URL doesn't resolve, you'll see a "media unavailable" notice — point the URLs at real assets to enable playback.

Adding new vCons

  • Drop new vcon-*.json files into this folder.
  • Run python3 _index.py to rebuild index.json.
  • Commit and push — GitHub Pages re-serves and the dashboard reflects the changes.

Running locally

Open a terminal in this folder and run python3 -m http.server 8000, then visit http://localhost:8000/. Opening the file directly via file:// will not work because browsers block fetch() from local files.
